## Civitai 영상 실물이 여기 없는 이유

이 저장소는 public이다. 선정된 Civitai 영상 20개(보라·네온 계열, 게시자 V3rseFX)는 사용 모델 13종의 `allowCommercialUse` 설정이 허용적이지만, 이는 **해당 모델로 새 결과물을 생성하는 것**에 대한 근거이지 타인이 게시한 공개 샘플 MP4를 재배포할 권리까지 보장하지 않는다. 따라서:

- 영상 실물(MP4 20개, 10초, 오디오 포함)과 썸네일은 **팀 구글드라이브로만 공유**한다.
- 이 저장소에는 조사본 PDF와 메타데이터(원본 링크·재현용 프롬프트)만 둔다.
- 정식 서비스 적용 전에는 원 업로더 허가를 받거나, 허용 모델로 자체 생성한 영상으로 교체한다.
