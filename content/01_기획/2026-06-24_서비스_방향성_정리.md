# 서비스 방향성 정리

정리일: 2026-06-30
목적: KX GenStudio가 어떤 서비스로 인식되어야 하는지 방향성을 정리한다.

## 1. 포지셔닝

KX GenStudio는 일반 이미지 생성 툴이 아니라 방송 제작자의 프리 프로덕션을 돕는 제작 보조 서비스다.

| 축 | 방향 |
| --- | --- |
| 대상 | 방송 PD, AD, 영상 기획자 |
| 상황 | 기획안, 콘셉트 회의, 발표 자료 준비 |
| 핵심 결과물 | 배경 시트, 캐릭터 시트, 키이미지, 짧은 영상 |
| 차별점 | 이미지 생성, 영상 생성, 라이브러리 흐름을 한 서비스에서 연결 |

## 2. 사용자 가치

| 가치 | 설명 |
| --- | --- |
| 빠른 시각화 | 글로 된 기획을 이미지와 영상 초안으로 빠르게 확인한다. |
| 반복 작업 감소 | 여러 AI 사이트를 오가는 시간을 줄인다. |
| 결과물 재사용 | 생성 결과를 라이브러리에 저장해 회의와 발표에 다시 활용한다. |
| 제작 맥락 반영 | 방송 장면, 구도, 조명, 분위기를 중심으로 프롬프트를 설계한다. |

## 3. 참고 서비스 관점

| 참고 서비스 | 참고 포인트 | A2팀 적용 방식 |
| --- | --- | --- |
| Higgsfield | 짧은 영상 생성 경험 | 3~15초 영상 생성 기준으로 검토 |
| Magnific | 고품질 이미지 결과 | 최종 후보 이미지 품질 기준 참고 |
| Krea | 빠른 생성 경험 | 실시간 편집은 제외하고 빠른 입력/결과 흐름만 참고 |

## 4. 서비스 톤

| 항목 | 방향 |
| --- | --- |
| UI 톤 | 작업 중심, 명확한 상태 표시, 복잡한 설명 최소화 |
| 문구 | 방송 제작자가 이해하기 쉬운 단어 사용 |
| 화면 흐름 | 생성, 결과 확인, 저장, 다시 조회가 끊기지 않게 구성 |
| 상태 표시 | 로딩, 실패, 빈 상태를 숨기지 않고 명확히 안내 |

## 5. 방향성 요약

KX GenStudio는 방송 제작자가 기획 단계에서 이미지를 만들고, 3x3 그리드 기반 영상을 생성하고, 결과물을 라이브러리에 모아 관리하는 제작 보조형 생성 AI 서비스다.
