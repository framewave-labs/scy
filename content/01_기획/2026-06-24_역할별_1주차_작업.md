5. 1주차 일자별 흐름
Day 1
PM: PRD, 평가 기준, MVP 범위 공유
UX/UI: IA 작성 시작
Frontend: 프로젝트 세팅
Backend 1: 이미지 API 호출 방식 확인
Backend 2: fal.ai 호출 환경 확인
Day 2
PM: 2주차 Pass 기준 확정
UX/UI: 생성 화면 와이어프레임 작성
Frontend: 기본 라우팅 및 생성 화면 UI 구성
Backend 1: 이미지 생성 테스트
Backend 2: 영상 submit 테스트
Day 3
PM: 라이브러리 저장 항목 정리
UX/UI: 라이브러리 화면 와이어프레임 작성
Frontend: 이미지 결과 영역 및 저장 버튼 UI 구성
Backend 1: 이미지 생성 API Route 초안 작성
Backend 2: Supabase 저장 구조 초안 작성
Day 4
PM: 중간점검용 시연 흐름 정리
UX/UI: 로딩, 실패, 빈 상태 UI 정리
Frontend: mock 데이터 기반 라이브러리 화면 구성
Backend 1: 이미지 응답 포맷 및 에러 구조 정리
Backend 2: 영상 status 조회 테스트
Day 5
PM: 1주차 점검 및 2주차 실행 계획 정리
UX/UI: 주요 화면 와이어프레임 공유
Frontend: 기본 화면 구조 공유
Backend 1: 이미지 API 검증 결과 공유
Backend 2: 영상 API 및 저장 구조 검토 결과 공유
