# A2팀 생성형 AI 통합 엔진 프로젝트 방향 정리

작성일: 2026-06-26
업데이트: 2026-07-22
작성자: 손찬용 PM

> 서비스명 표기 갱신 (2026-07-22): 작성 당시 임시명 `KX GenStudio`로 서술했으나, KX 엔터테인먼트(본사) 지정명 `GeNova`를 2026-07-20 수령해 표기를 교체했다. 방향 내용 자체는 변경 없음.

## 1. 문서 목적

이 문서는 A2팀 서비스가 어떤 사용자 경험을 지향하는지 정리한다.

PRD가 요구사항을 다루고, MVP 문서가 평가 기준을 다룬다면, 이 문서는 제품의 방향성과 흐름을 설명한다.

## 2. 핵심 방향

GeNova는 방송 제작자가 기획 단계에서 필요한 이미지를 만들고, 그 결과를 바탕으로 짧은 영상 초안을 생성하는 제작 보조 서비스다.

```text
배경 시트 생성
→ 캐릭터 시트 생성
→ 키이미지 생성
→ 3x3 스토리보드 그리드 이미지 1장 입력
→ 영상 프롬프트 1건 입력 (컷 순서 1=좌상단 → 9=우하단으로 서술)
→ 3~15초 영상 생성
→ 라이브러리 저장
```

> 영상 입력 표기 갱신 (2026-07-22): 작성 당시에는 "9개 씬 프롬프트 입력"으로 서술했으나, 2026-07-14 협의에서 영상 생성 입력이 **3×3 그리드 이미지 1장 + 프롬프트 1건**으로 확정돼 표기를 맞췄다. 흐름 자체는 변경 없음.

## 3. 이미지 생성 방향

| 생성물 | 역할 |
| --- | --- |
| 배경 시트 | 공간, 시간대, 조명, 분위기 기준을 잡는다. |
| 캐릭터 시트 | 인물의 외형, 의상, 표정, 기본 구도를 잡는다. |
| 키이미지 | 배경과 캐릭터 기준을 합쳐 대표 장면을 만든다. |

이미지 생성은 단순히 예쁜 이미지 한 장을 만드는 기능이 아니라, 이후 영상 생성과 발표 자료에 활용할 시각 기준을 만드는 단계다.

## 4. 영상 생성 방향

영상 생성은 9장의 이미지를 각각 넣는 구조가 아니다. 최신 기준은 3x3 스토리보드 그리드 이미지 1장과 scene 1~9 프롬프트를 함께 입력해 하나의 짧은 영상을 생성하는 방식이다.

| 항목 | 기준 |
| --- | --- |
| 입력 이미지 | 3x3 스토리보드 그리드 이미지 1장 |
| 입력 텍스트 | scene 1~9 프롬프트 |
| 영상 길이 | Higgsfield 기준 3~15초 |
| 결과 | 하나의 영상 결과물 |
| 향후 확장 | Seedance 2.5 30초 생성 가능 시 검토 |

## 5. 라이브러리 방향

라이브러리는 단순 저장소가 아니라 제작 과정에서 생성한 후보를 다시 비교하고 재사용하는 공간이다.

| 기능 | 방향 |
| --- | --- |
| 저장 | 이미지와 영상을 모두 저장한다. |
| 조회 | 최신순 카드 목록으로 확인한다. |
| 상세 | 프롬프트, 생성 타입, 모델, 생성일을 확인한다. |
| 재사용 | 저장된 이미지를 키이미지나 영상 생성 흐름에 다시 활용한다. |

## 6. 선택 기능 방향

역프롬프트는 방송 제작자가 레퍼런스 이미지를 바탕으로 비슷한 분위기의 이미지를 만들 수 있게 돕는 선택 기능이다.

이 기능은 S등급 도전 범위이며, A등급 핵심 기능을 방해하지 않는 선에서 진행한다.

```text
레퍼런스 이미지
→ 프롬프트 추출
→ 프롬프트 수정
→ 이미지 재생성
→ 라이브러리 저장
```

## 7. 방향성 한 줄 정리

A2팀은 이미지 생성, 3x3 그리드 기반 영상 생성, 라이브러리를 하나의 흐름으로 묶어 방송 제작자가 빠르게 시각 초안을 만들고 관리할 수 있는 서비스를 만든다.
