## 작성 기준

이 문서는 A2팀이 1주차 동안 진행한 팀 단위 업무를 정리하기 위한 중간보고서입니다.

2026.07.02까지 확인된 회의록, PRD, 마일스톤, IA 준비 문서를 기준으로 작성했으며, 2026.07.03 업무는 퇴근 후 공유되는 PM·BE·UX/UI 진행 현황을 반영해 v0.2에서 업데이트합니다.

### 07.02(목) - 마일스톤 확정 범위 정리 및 IA 협의 준비

**PM**

| 업무 | 결과 |
|------|------|
| 마일스톤 v3.1~v3.2 업데이트 | 완료 - BE 2인 분담, AWS 기준, 3주차 중반 FE+BE 1차 배포 반영 |
| PRD v0.1 업데이트 | 완료 - PM/Backend 마일스톤 확정사항 반영 |
| 백엔드 일정 및 작업 범위 확인 | 완료 - 이미지 담당 / 영상 담당 분담 기준 정리 |
| UX/UI 협의 준비 | 완료 - 07.03 IA 및 와이어프레임 협의 항목 정리 |
| 화면 구성 개인 메모 작성 | 완료 - MVP 화면 구조, 라이브러리 중심 흐름, 커뮤니티 제외 범위 정리 |

**주요 결정 사항**
- 프론트엔드 배포는 Vercel, 백엔드 배포는 AWS 기준으로 정리
- 문서의 API 명칭은 기능 단위 엔드포인트로 표현하고, 특정 프레임워크 Route 구조로 단정하지 않음
- BE 작업은 이미지 담당 / 영상 담당으로 분리
- 세부 AWS 서비스 조합, 데이터 모델, API 요청·응답 형식, 저장 구조는 BE 명세서 확정 후 반영
- 커뮤니티, 게시글, 댓글, 추천 기능은 MVP에서 제외하고 추후 논의 항목으로 분리
- 생성 결과물은 게시글이 아니라 작업물 또는 라이브러리 항목으로 표현

**BE**
- 팀 레포 생성 및 브랜치 전략 확정
- API 명세서 작업 진행 중
- DB 명세서 및 저장 구조 초안 작업 진행 중
- AWS 백엔드 프로젝트 구조 세팅 착수

**UX/UI**
- 각 팀 디자이너 소통 및 와이어프레임 방향 논의
- 디자인 시스템 방향과 주요 화면 구조 논의 준비
- vivago.ai 등 생성형 AI 서비스 레퍼런스 리서치 진행
