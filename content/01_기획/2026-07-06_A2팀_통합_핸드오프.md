## 1. 현재 한 줄 요약

A2팀 서비스는 **KX GenStudio**로 정리되어 있으며, 사용자가 이미지 생성, 3x3 스토리보드 기반 영상 생성, 미디어 라이브러리 흐름을 웹에서 단계적으로 사용할 수 있게 만드는 생성형 AI 제작 서비스다.

현재 1주차까지는 PRD, 마일스톤, IA 1차 정리본, API/DB 명세서, 회의록, 1주차 중간보고서, QA/리스크 기준이 정리되어 있고, 2026-07-06에는 결제/배포 운영안 3종(API 결제, 배포환경 결제, 개발자 전달사항)이 추가로 정리됐다. 2주차 시작 시점의 핵심은 **FE 복귀 후 개발 환경 세팅**, **M001~M006 이미지 생성·저장·라이브러리 흐름 착수**, **API 요청/응답 스키마 확인**, **IA 후속 안건 정리**, **결제 완료 및 API 키 실사용 가능 여부 확인**이다.

## 3. 제품 기준

### 3.1 서비스 정의

| 항목 | 현재 기준 |
| --- | --- |
| 프로젝트명 | 프로젝트 A: 생성형 AI 통합 엔진 |
| 서비스명 | KX GenStudio |
| 서비스 형태 | 웹 기반 생성형 AI 제작 서비스 |
| 핵심 가치 | 이미지 생성, 3x3 스토리보드 기반 영상 생성, 미디어 라이브러리를 하나의 흐름으로 연결 |
| 프론트엔드 배포 | Vercel 기준 |
| 백엔드 배포 | AWS 기준 |
| 이미지 생성 | GPT Image 2 기준 |
| 영상 생성 | B서비스에서 받은 3x3 스토리보드 이미지 1장 + 영상 프롬프트 기반 |

### 3.2 핵심 사용자 흐름

```text
홈 진입
→ 이미지 생성
→ 생성 결과 확인
→ 라이브러리 저장
→ 라이브러리 조회
→ 3x3 스토리보드 이미지와 영상 프롬프트 입력
→ 영상 생성 요청
→ 진행상태 조회
→ 영상 결과 확인
→ 라이브러리 저장 및 재조회
```

### 3.3 범위 경계

| 포함 | 제외 또는 후순위 |
| --- | --- |
| 이미지 생성, 결과 표시, 저장, 라이브러리 조회 | 커뮤니티, 게시글, 댓글, 추천 |
| 3x3 스토리보드 이미지 기반 영상 생성 | 9개 컷을 각각 따로 업로드하는 구조 |
| 이미지/영상 통합 라이브러리 | 전체 사용자 검색, 인기순, 공개 피드 |
| 방향별 레퍼런스 확보 | 실제 3D 모델 생성 또는 360도 회전 모델 |
| 역프롬프트 S등급 도전 | 2주차 Pass 전 조기 착수 |

생성 결과물 표현은 `게시글`이 아니라 `작업물` 또는 `라이브러리 항목`으로 통일한다.
