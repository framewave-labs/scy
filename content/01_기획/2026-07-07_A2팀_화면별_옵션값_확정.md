# A2팀 화면별 옵션값 확정
> 작성일: 2026-07-07 | 갱신: 2026-07-09 | 작성자: 손찬용 PM
> 목적: 와이어프레임에 들어갈 옵션값을 디자이너·BE가 한 곳에서 확인할 수 있도록 정리. 와이어프레임 링크와 함께 전달한다.
> **상태: 임시 통합본 — 최종 확정 아님.** `2026-07-08_A2팀_화면별_옵션값_손찬용안.md`(손찬용)과 `2026-07-08_속성값_확정안.md`(이슬기 PM, lsg)를 대조한 `2026-07-08_A2팀_옵션값_통합_제안.md`의 권고를 반영해 이 문서를 실사용 가능한 단일안으로 정리했다. 아래 "미해결" 표시 항목만 남으면 v1.0으로 승격한다.
> v0.1 대비 변경: 이미지 생성 화면 구조를 IA v0.4의 메뉴 A(초기 레퍼런스)·메뉴 B(키이미지) 2단계로 재구성. "결정 필요"였던 항목 대부분 값 채움.

## 1. 홈 화면

| 항목 | 값 | 상태 |
| --- | --- | --- |
| 히어로 메시지 | (항목 삭제) | 확정 — IA v0.4에서 온보딩/히어로 섹션 자체를 없애기로 결정, 로그인 후 핵심 기능 바로 노출 |
| 입력창 | 프롬프트 입력 → 칩(초기 레퍼런스 / 키이미지 / 영상) 선택 → 엔터 → 해당 메뉴로 이동, 프롬프트 자동 채움. 칩 미선택 시 기본값은 초기 레퍼런스(메뉴 A) | 확정 |
| 전체 구성 | 대시보드 중심, 좌측 사이드바 여부는 후속안건으로 별도 논의 중 | 참고 |

## 2. 이미지 생성 — 메뉴 A(초기 레퍼런스)

| 옵션 | 값 | 상태 |
| --- | --- | --- |
| 모델 | GPT Image 2 (고정, 선택 UI 불필요) | 확정 |
| 목적 | 캐릭터 / 배경 (선택에 따라 생성 프롬프트 기본값 분기) | 확정 |
| 품질 | 표준(medium, 기본) / 고화질(high, 옵션) | 확정 — 화면에 두 단계로 노출 |
| 비율 | Auto(기본) / 1:1 / 16:9 / 9:16 | 확정 |
| 최대 생성 수량 | 1회 요청당 1~4장 (+/- 스테퍼). 일일 한도는 API 비용 운영안과 연동해 별도 결정 | 확정(1회분) / **미해결(일일 한도)** |
| 레퍼런스 첨부 | Style + Character + Add, 최대 8장, PNG/WEBP/JPG 50MB 이하 | 확정 |
| AI 프롬프트 교정 | ON/OFF 토글, Claude Sonnet이 한국어 → 영어 최적화 프롬프트로 변환 | 확정(UI) — 실제 변환 레이어 연동 시점은 후속안건 3번(BE 검토 중)에 따름 |
| 캐릭터 다각도 출력 | 정면·측면·후면을 1장에 담아 출력, 프롬프트 엔지니어링으로 처리(BE 별도 작업 불필요) | 확정 |

## 3. 이미지 생성 — 메뉴 B(키이미지)

| 옵션 | 값 | 상태 |
| --- | --- | --- |
| 모델 | GPT Image 2 (고정) | 확정 |
| 목적 | 키이미지 캐릭터시트 (표시만, 메뉴 진입으로 확정) | 확정 |
| 품질 | 표준(medium) / 고화질(high) | 확정 |
| 비율 | Auto / 1:1 / 16:9 / 9:16 | 확정 |
| 수량 | 1장 고정, UI 미노출 | 확정 |
| 속성 선택 사이드바 | 파트1(성별·나이대·체형) / 파트2(아트 스타일·세계관) / 파트3(헤어) / 파트4(얼굴) / 파트5(의상·악세서리) — 전체 선택형, 필수/선택 구분 없음 | **잠정 채택** — 손찬용 검토 의견 미반영 상태 |
| 기본값 고정(UI 미노출) | 포즈: 정면+측면+후면 / 배경: 흰 배경 / 수량: 1장 | 확정 |
| AI 프롬프트 교정 | ON/OFF, Claude Sonnet 변환 | 확정(UI) — 위와 동일하게 실제 연동 시점은 후속안건 3번에 따름 |

## 4. 영상 생성 화면

| 옵션 | 값 | 상태 |
| --- | --- | --- |
| 모델 | Seedance 2.0(기본, 데모/고품질) / Kling 3.0(테스트/저비용) — 카드 형태로 노출 | 확정 |
| 길이 | 4~15초 (fal.ai 실제 지원 범위 기준 — 기존 "3~15초" 표기 오류 정정) | 확정 |
| 비율 | Seedance: Auto/16:9/9:16/1:1/21:9 — **4:3·3:4는 fal.ai 공식 문서에서 미확인이라 잠정 제외** / Kling: Auto/16:9/9:16/1:1 | **미해결** — BE가 fal.ai 콘솔에서 Seedance 4:3·3:4 지원 여부 확인 후 최종 확정 |
| 해상도 | 720p(기본, Fast 티어) / 1080p(옵션, Standard 티어) / 4K(시연 전용, 별도 4K 모델) | 확정 — fal.ai 문서로 직접 확인, "BE 확인 필요" 해소 |
| Bitrate | High 고정, UI 미노출. BE 환경변수(demo/prod/showcase)로 제어 | 확정 |
| 스토리보드 입력 | B서비스가 제공하는 9컷 3×3 그리드 이미지 1장 업로드 | 확정 |
| AI 프롬프트 교정 | ON/OFF, Claude Sonnet 변환 | 확정(UI) — 실제 연동 시점은 후속안건 3번에 따름 |
