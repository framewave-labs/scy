# A2팀 프롬프트 가이드 초안 갱신 제안
> 작성일: 2026-07-08 | 작성자: scy
> 목적: PD가 2026-07-07 확정한 키이미지 정의를 `2026-06-24_프롬프트_가이드_초안.md`에 반영하는 안을 제안한다.
> 관련 문서: `2026-06-24_프롬프트_가이드_초안.md`(갱신 대상), `2026-07-07_A2팀_키이미지_정의_PD확인.md`(근거), framewave-docs `docs/pd-key-image-template-20260707` PR(이미 최종본에는 반영, 머지 대기 중)

## 1. 왜 갱신이 필요한가

framewave-docs의 "최종 프롬프트 가이드"는 이미 PD 확정 기준으로 갱신되어 PR로 올라가 있다(개발자 전달 완료 후 머지 예정). 하지만 그 PR의 소스 문서로 명시된 `2026-06-24_프롬프트_가이드_초안.md`(이 저장소) 자체는 아직 옛 정의를 그대로 담고 있어, **원본 소스와 통합본이 서로 어긋난 상태**다.

현재 초안 2.2/2.3은 "캐릭터 시트"와 "키이미지"를 별도 산출물로 다룬다:

| 절 | 현재 내용 |
| --- | --- |
| 2.2 캐릭터 시트 | 인물정보/의상/표정/헤어소품/기준포즈/시각스타일 |
| 2.3 키이미지 | 장면목적/배경기준/인물기준/구도/조명/감정/촬영스타일 — **캐릭터+배경이 결합된 장면** 프롬프트로 정의 |

PD 확정(07-07)에 따르면 키이미지는 캐릭터+배경 통합 장면이 아니라 **캐릭터 1명에 대한 고정 레퍼런스 시트**(턴어라운드+표정+액세서리+키컬러)이며, 캐릭터 시트와 사실상 동일한 산출물이다. 즉 지금의 2.2/2.3 절 구분 자체가 낡았다.

## 2. 제안하는 변경

### 2.1 절 구조

- 2.2 "캐릭터 시트" → 2.2 "**캐릭터 시트(=키이미지)**"로 병합
- 기존 2.3 "키이미지" 절은 삭제한다. (장면 연출은 키이미지가 아니라 생성 시점 프롬프트 텍스트로 지시하는 것이 PD 확정 방향이므로, 별도 절이 필요 없다)
- 2.1 배경 시트 절 첫 줄에 한 문장 추가: "360도(다각도) 커버리지 불필요 — 특징적인 요소만 담으면 나머지는 자연스럽게 생성된다."

### 2.2 새 2.2절 제안 본문 (framewave-docs 최종본과 동일 기준)

> 한 이미지 안에 턴어라운드(정면/측면/후면), 표정 배리에이션, 액세서리·소품 디테일, 스타일·키컬러를 전부 담는다. 이후 생성 요청에 이 이미지를 계속 참조로 넣어 일관성을 유지한다.

**간단형(텍스트 한 줄, 기존 형식 유지)**

```text
[인물 정보] [의상] [표정] [헤어/소품] [기준 포즈] [시각 스타일]
```

**PD 제공 표준형 (2단 구성 — 캐릭터 키이미지에 권장)**

Part 1은 이미지 전체를 설명하는 자연어 문단, Part 2는 아래 필드로 구성된 구조화 블록이다.

| 필드 | 내용 |
| --- | --- |
| SUBJECT | 인물 기본 정보, 헤어/외형, 표정 배리에이션, 포즈(턴어라운드 각도) |
| CLOTHING & STYLING | 의상 아이템별 소재·색상(헥스코드 포함) |
| ACCESSORIES | 액세서리별 소재·색상 |
| SCENE | 장면/레이아웃 유형, 핵심 요소 |
| BACKGROUND | 배경 요소 (그리드, 텍스트 블록, 컬러 스와치 등) |
| LIGHTING | 조명 방식 |
| TECHNICAL | 촬영/렌더링 기술 정보 (해당 없으면 N/A) |
| COMPOSITION | 구도 방식 |
| STYLE | 아트 스타일, 주요 색상 |
| ASPECT RATIO | 출력 비율 |
| CONSTRAINTS - Must keep | 반드시 유지해야 할 요소 |
| CONSTRAINTS - Avoid | 피해야 할 것 |

전체 예시(미라 한 캐릭터 원문)는 `2026-07-07_A2팀_키이미지_정의_PD확인.md`에 있으므로 이 초안에는 표만 넣고 예시는 링크로 대체한다.

### 2.3 건드리지 않는 부분

- 3장(영상 프롬프트 구조), 4장(영상 예시), 5장(역프롬프트), 6장(피해야 할 작성 방식) — PD 확정과 무관하므로 그대로 둔다.
- `A2팀_프롬프트가이드_v1_20260625.md`(lsg, 키워드 모음·테스트 세트 20개) — 키이미지 정의를 다루지 않으므로 갱신 불필요.
