1. 영상 내부 처리 방식 비교 결과
2. 영상 진행 표시 방식
3. Kling 비율 UI
4. Kling 오디오 사용 여부
5. 프롬프트 향상 결과의 사전 노출·수정 여부
6. 라이브러리 필터·정렬 노출·동작

9. 삭제·탈퇴 정책

기능명세서는 최종 IA의 `IA ID`를 기준키로 작성하고, 위 9개 항목은 확정값처럼 기입하지 않는다. 협의 결과가 나오면 IA 후속안건과 기능명세서를 같은 작업 단위에서 함께 갱신한다.

## 2. 영상 파이프라인 — 오늘 논의로 굳어진 이해

- I2V 계열 모델은 입력 이미지를 "첫 프레임 그 자체"로 쓰는 구조라, 3×3 그리드 1장을 통째로 넣으면 그리드가 화면에 그대로 노출될 위험이 있다는 게 이슬기 PM 조사로 확인됨.
- 이 문제의 대응으로 **그리드를 9장으로 슬라이스(단순 crop)하는 전처리**가 필요하다는 게 원래 가설이었으나, BE가 실제로 Kling에 "시작 프레임(컷1) + 끝 프레임(컷9) + 3×3 그리드 전체를 레퍼런스로" 넣는 현행 방식으로 이미 테스트를 진행 중임을 확인 — 이 방식은 별도 논의 없이 **인정하고 유지**하기로 함.
- Seedance는 `reference-to-video` 엔드포인트(레퍼런스 최대 9이미지+3영상+3오디오, `@ImageN` 참조)로 9장을 분리 입력하는 방식이 유력하나, 아직 테스트 전.
- BE가 **4초 길이로 9컷을 다 보여주려다 컷 유지가 안 된다는 의견**을 냄 → 원인은 컷당 시간이 4초/9컷=0.4초로 물리적으로 부족했기 때문. **컷당 1.5~2초를 확보하는 조건(15초 기준)으로 재설계** 합의.
- BE가 "애초 기획이 3×3 그리드 그대로 넣는 것이었으니 슬라이스 없이 그리드 1장 그대로도 테스트해보자"는 의견을 제시 → **일리 있는 제안으로 받아들여 A/B 비교 테스트에 포함**하기로 함(슬라이스 전처리가 애초에 불필요할 수도 있다는 가설 검증 겸함).

### 2.1 최종 합의된 테스트 시나리오 (BE에 전달 예정/전달 중)

공통: 번호 오버레이 없는 3×3 그리드 1장, 컷별 프롬프트 준비, 해상도 720p, 매 건 생성 소요 시간 기록.

| # | 내용 | 길이 | 예상 비용 |
| --- | --- | --- | --- |
| 1 | Seedance reference-to-video — **3×3 그리드 1장 그대로** 입력 | 15초 | 약 $4.6 |
| 2 | Seedance reference-to-video — **9장 슬라이스** 입력 (같은 스토리보드) | 15초 | 약 $4.6 |
| 3 | Kling 현행 방식(시작=컷1 / 끝=컷9 / 그리드 레퍼런스), 같은 스토리보드 | 15초 | 약 $1.3 |

판정 기준: 순서 유지 여부 / 컷 식별 개수 / 전환 자연스러움, 3건을 나란히 비교.

**갈림길**:
- 1번이 2번만큼 잘 나오면 → 슬라이스 개발 없이 그리드 1장 방식으로 확정 (BE 작업량 최소)
- 2번이 확실히 좋으면 → 슬라이스 전처리(crop + `@ImageN` 프롬프트 변환) 개발 진행
- Kling(3번)이 둘 다 이기면 → Kling 현행 방식을 기본으로 검토

**미완료**: 이 시나리오는 오늘 대화에서 확정했으나 **아직 BE에 정식 전달 메시지로 발송하지 않은 상태**로 보임 — 다음 세션에서 발송 여부 확인 필요.

- "AI 프롬프트 교정" 토글이 이미지·영상 화면 둘 다 있음 → 영상에는 없어야 함(오늘 확정), 이미지 쪽 명칭도 "프롬프트 향상"으로 변경 필요
- 이미지 품질에 4K, 수량 최대 4장 노출 → 확정값은 표준/고화질 2단, 수량은 목적별 자동
- 이미지 생성이 "목적 선택" 드롭다운 하나로 통합 → 메뉴 A/B 2단계 분리가 확정 구조
- 크레딧 320·무료 크레딧 +50 노출 → 일일 한도는 UI 미노출이 확정
- 영상 품질 드롭다운(고화질 1920×1080) → Kling은 1080p 고정 읽기 전용, Seedance만 선택

## 7. 참고 — 오늘 확인한 외부 API 스펙 (fal.ai)

- `fal-ai/kling-video/v3/standard/image-to-video`: `start_image_url`(필수)·`end_image_url`(선택)·`prompt`·`duration`("3"~"15")·`generate_audio`(기본 true)·`multi_prompt`(최대 6샷, 총 15초 제한이라 9컷과 매핑 불가)·비율/해상도 파라미터 없음(1080p 고정).

- 가격(15초 기준): Kling Standard 1080p $1.26(오디오 OFF)/$1.89(ON) · Seedance 720p $4.55 · Seedance 1080p $10.23.
