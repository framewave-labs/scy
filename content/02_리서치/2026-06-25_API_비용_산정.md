# API 비용 산정

정리일: 2026-06-29  
원본 위치: Notion `Kx 인턴쉽 개인파일 정리` / `작업일정 관리` DB 연결 문서

프로젝트명: 생성형 AI 통합 엔진
목적: 프로젝트 기간 동안 필요한 AI 이미지·영상 생성 API 및 저장 비용을 미리 산정한다.
1. 비용 산정 기준
본 프로젝트는 사용자가 프롬프트를 입력하면 이미지를 생성하고, 생성된 키이미지와 샷별 프롬프트를 기반으로 영상을 생성하는 서비스이다.
영상 생성은 일반 생성 / 스토리보드 생성으로 기능을 나누지 않고, 하나의 샷 기반 영상 생성 방식으로 통합한다.
1샷 생성: 일반 영상 생성
2~9샷 생성: 스토리보드형 영상 생성
수정이 필요한 경우 전체 재생성이 아니라 해당 샷만 재생성
즉, 기능은 하나로 두고 비용은 샷 수와 영상 길이에 따라 계산한다.

9. 최종 정리
본 프로젝트의 영상 생성은 샷 기반 생성 방식으로 통합한다.
일반 생성과 스토리보드 생성을 별도 기능으로 나누기보다, 사용자가 선택한 샷 수에 따라 비용이 달라지는 구조로 산정한다.
A등급 목표 예산은 약 $250~300이며, S등급 도전까지 포함하면 약 $400~500 수준으로 본다.
최종 권장 예산은 실패 호출, 발표 직전 재생성, Supabase 저장 비용을 고려해 약 $500으로 잡는다.
