## 논의 내용

### 1. PM 마일스톤 작업

오늘 PM들은 마일스톤 작업을 진행했다.

백엔드 일정은 백엔드 담당자와 맞춰 확정했다. 디자이너 파트는 2026-07-03에 추가로 이야기해본 뒤 확정할 예정이다. 프론트엔드 파트는 담당자가 월요일에 합류한 뒤 이야기해볼 예정이다.

마일스톤 기준으로 이후 작업 순서와 담당별 우선순위를 계속 정리할 필요가 있다.

### 2. Backend 작업

백엔드 담당자는 API 명세서 작업을 진행 중인 것으로 보인다.

DB 명세서 작업도 함께 진행 중인 것으로 보인다. 두 문서 모두 아직 완료 상태라기보다 작업 중인 상태로 기록한다.

다음 확인 시점에는 API 명세와 DB 명세가 프론트엔드와 연결 가능한 수준인지 확인해야 한다.

### 3. UX/UI 작업

디자이너는 회의 종료 후 합류했다.

IA 작업과 와이어프레임은 2026-07-03에 진행하기로 했다. 화면 구성은 PM과 디자이너가 함께 논의할 예정이다.

내일 논의에서는 생성 화면, 결과 화면, 라이브러리 화면의 구조를 우선 확인하는 것이 필요하다.
