## 논의 내용

### 1. API 결제 진행 상황

2026-07-03 공지대로 API와 AI 에이전트 비용 결제는 오늘(월) 논의 이후 확정하는 일정으로 진행 중이다.

이슬기 PM이 KX엔터테인먼트에 오프라인으로 방문해 각 팀 PM들과 함께 PD님과 회의를 진행하고 있으며, 회의 후 전달받는 내용대로 API 결제를 마무리할 예정이다.

### 2. 개발자 API 비용 사용 논의

팀 개발자들은 API 관련 비용을 어떻게 사용할지 논의 중이다. 백엔드의 EC2/S3 요금산정은 2026-07-03 공지에서 월요일 협의 항목으로 남겨둔 건으로, 상세 예산은 백엔드 담당자가 산정해 전달하기로 했다.

### 3. 팀원 계정 정보 수집 완료

2026-07-03 결제 공지에서 요청한 팀원별 계정 정보가 팀 채널을 통해 모두 수집됐다.

- Frontend 김은재: 클로드·Vercel 계정 회신 완료 (복귀 당일 회신)
- Backend 송영범·김유신: 클로드 계정 회신 완료, AWS 계정은 송영범 계정으로 통일해 사용하기로 결정
- UX/UI 이예린: 클로드·피그마 계정 회신 완료 (동일 계정)

계정 상세(이메일 등)는 보안상 문서에 기록하지 않고 팀 채널 기준으로 관리한다.

### 4. 고정 구독 구성 확정

KX_전체비용_통합 시트(A2) 기준으로 고정 구독 구성이 확정됐다.

| 항목 | 구성 | 월 비용 |
| --- | --- | --- |
| Claude Team | 4석 | $120 |
| Figma Professional | 2석 (FE 1 + UX/UI 1) | $40 |
| Vercel Pro | 1석 | $20 |
| AWS EC2 | 예비 예산 (상세는 BE 산정 예정) | $100 |

저장 구조는 AWS EC2 + S3로 확정되어 Supabase는 구독 목록에서 제외했다. 사용량 기반 API 비용을 포함한 총 예상 비용은 $359.92(약 55.8만원) 기준이다.

### 5. FE 복귀 및 2주차 착수

FE 김은재가 1주차 결근 후 오늘 복귀했다. 2주차 계획(마일스톤 v3.2 기준)에 따라 레포 클론, 로컬 실행, 라우팅·공통 컴포넌트 구조 확인부터 진행한다.

## 결정 사항

- API 결제는 KX엔터테인먼트 오프라인 PM 회의(이슬기 PM 참석) 후 전달받는 내용대로 마무리한다.
- AWS 계정은 송영범 계정으로 통일해 사용한다.
- 고정 구독은 Claude Team 4석, Figma Professional 2석, Vercel Pro 1석으로 확정한다.
- 저장 구조는 AWS EC2 + S3 기준으로 하고 Supabase는 제외한다.
- AWS 상세 예산은 백엔드 담당자가 산정해 전달한다.
- 팀원 계정 상세 정보는 문서에 기록하지 않고 팀 채널 기준으로 관리한다.
