# A2팀 PM 사전 체크리스트

작성일: 2026-06-26
정리일: 2026-06-30
작성자: 손찬용 PM

## 1. 문서 성격

이 문서는 6월 30일 팀 작업 시작 전 PM이 확인해야 했던 사전 체크리스트이자 이전 버전 기록이다.

현재 기준 문서는 PRD, MVP 평가 기준, 작업범위 정리 문서로 분리되었으므로 이 문서는 보관용으로 둔다.

## 2. 체크 항목

| 항목 | 확인 상태 | 현재 관리 문서 |
| --- | --- | --- |
| 이미지 생성 모델 | GPT Image 2.0 기준으로 정리 | PRD |
| 2주차 Pass 흐름 | 이미지 생성, 저장, 조회 | MVP 기능 범위 및 평가 기준 |
| 영상 생성 방식 | 3x3 그리드 이미지 1장 + 9개 씬 프롬프트 | 프로젝트 방향 정리 |
| 역프롬프트 | 선택 기능으로 분리 | PRD / MVP 기능 범위 |
| 비용 산정 | 수요일까지 기준안 준비 | API 비용 산정 |
| 배포 방식 | 프론트 Vercel Hobby, 백엔드 AWS EC2/S3 | 엔지니어링 검토 메모 |

## 3. PM 우선순위

1. 팀원이 2주차 Pass 기준을 같은 의미로 이해하게 한다.
2. 개발팀이 API, DB, 배포 결정을 내릴 수 있게 질문을 정리한다.
3. UX/UI가 화면 흐름과 상태 UI를 설계할 수 있게 기능 범위를 명확히 한다.
4. 비용 산정은 영상 길이와 호출 수 기준으로 다시 계산한다.
5. 역프롬프트는 선택 기능임을 반복해서 공유한다.

## 4. 보관 이유

이 문서는 초기 PM 사고 흐름을 남기기 위한 문서이며, 중복 요약을 막기 위해 `summary_target: false`로 둔다. 최신 평가 기준과 작업 범위는 MVP 평가 기준 문서와 작업범위 정리 문서를 따른다.
