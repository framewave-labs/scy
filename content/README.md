## 요약 대상 기준

| 값 | 의미 |
| --- | --- |
| `summary_target: true` | 현재 기준 문서 또는 통합본에 반영해야 할 문서 |
| `summary_target: false` | 원본 보관, 빈 문서, 과거 초안, 참고자료 |

## 버전 관리 기준

문서 버전관리 규칙은 저장소 루트의 `README.md`를 기준으로 합니다.

초안과 과거 작업본은 삭제하지 않고 `scy`에 남기되, 최신본만 통합 요약에 반영되도록 `status`, `version`, `document_group`, `current`, `summary_target`, `superseded_by`를 관리합니다.
